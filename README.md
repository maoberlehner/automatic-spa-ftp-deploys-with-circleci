# Automatic SPA FTP Deploys with CircleCI

[![Donate](https://img.shields.io/badge/Donate-PayPal-blue.svg)](https://paypal.me/maoberlehner)

This is an example project for the following article: [Automatic SPA FTP Deploys with CircleCI](https://markus.oberlehner.net/blog/automatic-spa-ftp-deploys-with-circleci/)

## Build Setup

```bash
# Serve with hot reload.
yarn dev

# Build for production with minification.
yarn build

# Serve SSR production build.
yarn start

# Generate static pre-rendered build.
yarn generate
```
